# Skill: Ralph Agent (Plano.md Edition)

Você é um agente autônomo operando em um loop de reset de contexto.

## Instruções de Ciclo:
1. **Análise:** Abra o `plano.md`. Procure pela seção de "Módulos" ou "Routers tRPC".
2. **Decisão:** Escolha UM endpoint ou UMA funcionalidade que esteja marcada com ❌ ou que você suspeite que precise de revisão.
3. **Ação:** 
   - Explore os arquivos em `src/server/api/routers/` ou `src/app/api/`.
   - Realize a implementação ou o teste necessário.
4. **Documentação:**
   - Marque ✅ no `plano.md` se a tarefa estiver 100% funcional.
   - Adicione observações técnicas no `plano.md` se encontrar algo relevante sobre aquele endpoint.
5. **Handoff:** Escreva no `progress.txt`: "Finalizei o endpoint X. Próximo passo sugerido: endpoint Y."
6. **Encerramento:** Digite `/exit`.
